# Special Olympics Extension

Special Olympics Liechtenstein WordPress extension plugin providing member (CPT) management, trainings/events administration, attendance tracking, payroll/export tools, and related taxonomies/ACF integrations.

## Installation

1. Copy the `special-olympics-extension` folder into `wp-content/plugins/`.
2. Activate the plugin in the WordPress admin.

## Notes

- This plugin depends on **Advanced Custom Fields (ACF)**.
- Releases are tracked in [`CHANGELOG.md`](CHANGELOG.md).
